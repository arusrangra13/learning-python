# 🐍 Learning Python

A personal repository documenting my journey of learning Python — from the basics to more advanced concepts.

## 📚 What's Inside

- **Basics** — Variables, data types, operators, input/output
- **Control Flow** — if/else, loops (for, while), break/continue
- **Functions** — Defining functions, arguments, return values, lambda
- **Data Structures** — Lists, tuples, dictionaries, sets
- **Object-Oriented Programming** — Classes, objects, inheritance, polymorphism
- **File Handling** — Reading and writing files
- **Modules & Packages** — Importing, creating modules, using pip
- **Error Handling** — try/except, custom exceptions
- **Projects** — Small hands-on projects to apply concepts

## 🛠️ Requirements

- Python 3.x
- No external dependencies required (standard library used)

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/arusrangra13/learning-python.git

# Navigate into the folder
cd learning-python

# Run any script
python filename.py
```

## 🎯 Goal

To build a strong foundation in Python programming through consistent practice and mini-projects.
